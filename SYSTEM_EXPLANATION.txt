2. BAGAIMANA CARA KERJA SISTEM?
================================================================================

PROSES REGISTRASI:
1. Karyawan berdiri di depan kamera
2. Sistem mendeteksi wajah menggunakan Haar Cascade
3. Sistem mengekstrak fitur wajah (histogram)
4. Fitur disimpan dengan nama karyawan di database

PROSES ATTENDANCE:
1. Kamera menangkap video real-time
2. Sistem mendeteksi wajah di setiap frame
3. Ekstrak fitur dari wajah yang terdeteksi
4. Bandingkan dengan database wajah yang terdaftar
5. Jika cocok (confidence > threshold), catat kehadiran
6. Simpan data ke file CSV dengan timestamp

ANTI-DUPLICATE SYSTEM:
- Sistem mengecek apakah karyawan sudah absen hari ini
- Jika sudah, sistem akan menolak absen ganda
- Cooldown 3 detik antar recognition untuk efisiensi

6. ALGORITMA YANG DIGUNAKAN
================================================================================

ğŸ¯ FACE DETECTION - HAAR CASCADE:
- Menggunakan pre-trained classifier dari OpenCV
- Mendeteksi objek berbentuk wajah dalam gambar
- Fast dan efficient untuk real-time processing
- Akurasi: ~90% dalam kondisi normal

Cara kerja:
1. Scan gambar dengan sliding window
2. Hitung Haar features di setiap window
3. Klasifikasi menggunakan cascade classifier
4. Return koordinat bounding box wajah

ğŸ“ˆ FEATURE EXTRACTION - HISTOGRAM:
- Mengkonversi wajah ke grayscale
- Resize ke ukuran standar (100x100)
- Hitung histogram intensitas pixel
- Normalize histogram untuk konsistensi

Formula histogram:
H(i) = jumlah pixel dengan intensitas i
H_norm(i) = H(i) / total_pixels

ğŸ” FACE MATCHING - CORRELATION COEFFICIENT:
- Bandingkan histogram dua wajah
- Hitung correlation coefficient
- Range: -1 (tidak mirip) sampai 1 (identik)
- Threshold: 0.7 (dapat disesuaikan)

9. TROUBLESHOOTING
================================================================================

âŒ MASALAH UMUM DAN SOLUSI:

1. CAMERA TIDAK TERDETEKSI:
   Gejala: "Error: Tidak dapat mengakses webcam"
   Solusi:
   - Cek koneksi camera
   - Tutup aplikasi lain yang menggunakan camera
   - Ganti index camera: cv2.VideoCapture(1)
   - Install driver camera terbaru

2. FACE DETECTION TIDAK AKURAT:
   Gejala: Wajah tidak terdeteksi atau false positive
   Solusi:
   - Perbaiki pencahayaan
   - Posisi wajah tegak lurus ke camera
   - Jarak optimal: 50-100cm
   - Adjust detection parameters

3. FACE RECOGNITION SALAH:
   Gejala: Mengenali orang yang salah
   Solusi:
   - Registrasi ulang dengan foto lebih jelas
   - Turunkan threshold confidence
   - Tambah multiple foto per person
   - Perbaiki kualitas camera

4. PERFORMANCE LAMBAT:
   Gejala: FPS rendah, lag
   Solusi:
   - Kurangi resolusi camera
   - Optimize detection area
   - Gunakan threading
   - Upgrade hardware

5. DEPENDENCIES ERROR:
   Gejala: Import error, module not found
   Solusi:
   - Reinstall virtual environment
   - Update pip: pip install --upgrade pip
   - Install ulang: pip install -r requirements.txt
   - Cek Python version compatibility
